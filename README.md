# CRACKLIX Neural Exam Engine

This project is the official codebase for the CRACKLIX Neural Exam Engine, a high-performance mock exam platform for Punjab competitive exams.

## System Goal

Create a high-performance Punjab competitive exam mock platform for:

* PSSSB
* PSTET
* CTET
* ETT Cadre
* Master Cadre
* PSEB Lecturer
* Excise Inspector
* Clerk
* Patwari
* Jail Warder
* Punjab Police
* PSPCL
* PSTCL

## Core Philosophy

Keep the app ultra optimized and lightweight. Avoid infinite loops, heavy animations, auto-refresh loops, unnecessary useEffect renders, memory leaks, and excessive shadows/blurs.

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build

```bash
npm run build
npm run start
```

### Type Checking

```bash
npm run typecheck
```

### Linting

```bash
npm run lint
```
