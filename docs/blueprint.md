# **App Name**: Cracklix

## Core Features:

- Unified Auth & Role System: Secure authentication using Firebase Auth with Google and email/password support, incorporating specific roles like student and admin.
- Gamified Student Dashboard: A personal command center for students to track performance metrics including total XP, active login streaks, and historical exam progress.
- AI Performance Coach tool: A generative AI powered tool that uses reasoning to analyze your quiz results and provide a custom study plan based on identified weaknesses.
- Persistent Study Profiles: Store and retrieve user data, learning stats, and XP points reliably using a Firestore database backend configured with the cracklixapp project.
- Dynamic Exam Engine: A realtime testing environment that manages timed sessions and automatically calculates scores and rewards.
- Global Leaderboard: Real-time ranking of students based on XP and streak achievements to drive competition and engagement.
- Fluid Motion Feedback: High-performance UI interactions and feedback loops powered by Framer Motion for a premium digital experience.

## Style Guidelines:

- The visual palette is anchored in a high-performance Dark mode. Primary color: Electric Blue (#3380FF). Background: Slate Deep Blue (#0F1115). Accent: Cyan Pulse (#1AE5E5).
- Typography features a pairing of 'Space Grotesk' for headlines to evoke a tech-focused feel and 'Inter' for body text for optimal readability across complex exam content.
- Lean, sharp line icons from Lucide-React to maintain a clean, high-precision aesthetic.
- A responsive, grid-based dashboard with rounded corners (24px to 32px) and generous white space to prevent cognitive overload during study sessions.
- Smooth, meaningful micro-interactions and transitions using Framer Motion to provide tactile feedback during score updates.
